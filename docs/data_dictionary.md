# 📊 Data Dictionary

This document describes the structure of datasets generated by the pipelines.

---

## Data Conventions

- Dates follow ISO format (YYYY-MM-DD)
- Monetary values are expressed in Brazilian reais (BRL)
- Decimal separator in CSV files uses ','
- Column separator uses ';'

----

## 01_bcb_indicadores_economicos.csv

Macroeconomic indicators collected from Banco Central do Brasil.

| Column | Type  | Description                          |
| ------ | ----- | ------------------------------------ |
| data   | date  | Observation date                     |
| selic  | float | Brazilian base interest rate (SELIC) |
| ipca   | float | Official inflation index (IPCA)      |
| dolar  | float | USD/BRL exchange rate                |

Source: Banco Central do Brasil (SGS API)

## 03_yfinance_eventos_raw.csv

Corporate actions collected from Yahoo Finance.

These events represent cash distributions and capital structure changes that may affect historical price behavior.

### Schema

| Column      | Type   | Description                    | Source        | Notes                  |
| ----------- | ------ | ------------------------------ | ------------- | ---------------------- |
| ticker      | string | Stock ticker symbol            | Yahoo Finance | B3 ticker              |
| date        | date   | Event date                     | Yahoo Finance | ISO format             |
| dividend    | float  | Dividend amount paid per share | Yahoo Finance | In BRL                 |
| stock_split | float  | Stock split ratio              | Yahoo Finance | Example: 2 = 2:1 split |

### Example Record

| ticker | date       | dividend | stock_split |
| ------ | ---------- | -------- | ----------- |
| PETR4  | 2023-05-15 | 1.23     | 0           |

## 04_yfinance_info_raw.csv

Company metadata and fundamental indicators retrieved from Yahoo Finance.

This dataset contains company profile information and financial metrics useful for screening and fundamental analysis.

Because the Yahoo Finance API returns a large dictionary of attributes, the number of columns may exceed **100 depending on availability**.

### Core Identification Fields

| Column   | Type   | Description             | Source        |
| -------- | ------ | ----------------------- | ------------- |
| ticker   | string | Stock ticker            | Yahoo Finance |
| longName | string | Full company name       | Yahoo Finance |
| sector   | string | Economic sector         | Yahoo Finance |
| industry | string | Industry classification | Yahoo Finance |
| country  | string | Country of headquarters | Yahoo Finance |


### Market Data Metrics

| Column            | Type  | Description                 |
| ----------------- | ----- | --------------------------- |
| marketCap         | float | Total market capitalization |
| enterpriseValue   | float | Enterprise value            |
| sharesOutstanding | int   | Total shares outstanding    |


### Valuation Metrics

| Column              | Type  | Description                 |
| ------------------- | ----- | --------------------------- |
| trailingPE          | float | Price/Earnings ratio (TTM)  |
| forwardPE           | float | Forward Price/Earnings      |
| priceToBook         | float | Price to book ratio         |
| enterpriseToRevenue | float | Enterprise value to revenue |

### Profitability Metrics

| Column           | Type  | Description            |
| ---------------- | ----- | ---------------------- |
| profitMargins    | float | Net profit margin      |
| grossMargins     | float | Gross margin           |
| operatingMargins | float | Operating margin       |
| returnOnAssets   | float | Return on assets (ROA) |
| returnOnEquity   | float | Return on equity (ROE) |

### Risk Metrics

| Column       | Type  | Description                   |
| ------------ | ----- | ----------------------------- |
| beta         | float | Volatility relative to market |
| debtToEquity | float | Debt-to-equity ratio          |

### Liquidity Metrics

| Column       | Type  | Description                          |
| ------------ | ----- | ------------------------------------ |
| currentRatio | float | Current assets / current liabilities |
| quickRatio   | float | Quick liquidity ratio                |

### Notes

- The dataset structure depends on the Yahoo Finance API response.
- Some columns may contain missing values (**NaN**) depending on company coverage.
- All metrics are returned exactly as provided by Yahoo Finance without transformation.

## 05_yfinance_balancos_raw.csv

Annual financial statements extracted from Yahoo Finance.

This dataset represents **Income Statement data (DRE)** for B3 companies.

Each row corresponds to **one company in one fiscal year**.

---

### Schema

| Column           | Type   | Description             | Source        | Notes                       |
| ---------------- | ------ | ----------------------- | ------------- | --------------------------- |
| ticker           | string | Stock ticker            | Yahoo Finance | B3 symbol                   |
| fiscalDateEnding | date   | Fiscal year end date    | Yahoo Finance | Annual reporting period     |
| totalRevenue     | float  | Total company revenue   | Yahoo Finance | In BRL                      |
| grossProfit      | float  | Gross profit            | Yahoo Finance | Revenue minus cost of goods |
| operatingIncome  | float  | Operating income (EBIT) | Yahoo Finance |                             |
| netIncome        | float  | Net income              | Yahoo Finance | Final company profit        |

---

### Example Record

| ticker | fiscalDateEnding | totalRevenue | grossProfit | operatingIncome | netIncome   |
| ------ | ---------------- | ------------ | ----------- | --------------- | ----------- |
| VALE3  | 2022-12-31       | 208000000000 | 91000000000 | 62000000000     | 41000000000 |
